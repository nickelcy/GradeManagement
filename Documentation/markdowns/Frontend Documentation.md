# Frontend Documentation

Overview
This guide explains how to run the GradeManagement frontend locally.

Prerequisites
- Node.js (run `node -v` to check)
- npm (run `npm -v`)

Setup
```bash
cd Frontend
npm install
```

Environment
- The frontend uses `VITE_API_BASE_URL`.
- If unset, it defaults to `http://localhost:8001/api`.

Run the dev server
```bash
npm run dev
```

Build
```bash
npm run build
```

Preview
```bash
npm run preview
```
